Pro Multi-Tab NotepadPro Multi-Tab Notepad は、Python と CustomTkinter を使用して構築された、モダンで高機能なマルチタブ型テキストエディタです。日々のテキスト編集から、プログラミング（Python, HTML, CSS, JavaScript）まで、快適な作業環境を提供します。🌟 主な特徴モダンな UI: ダークモードおよびライトモードをサポート。洗練されたデザインで長時間の作業をサポートします。マルチタブ管理: 複数のファイルを同時に開き、タブ切り替えでスムーズに編集可能です。コーディング支援:シンタックスハイライト: Python, HTML, CSS, JS に対応。行番号表示: リアルタイムで更新される行番号。グリッド表示: 水平・垂直のガイド線（罫線）を表示可能。現在行強調: カーソルがある行をハイライトします。強力な検索機能: リアルタイムのハイライト表示と、前後へのナビゲーション機能を搭載。多言語対応 (i18n): 日本語と英語の表示を自由に切り替え可能。スマートなディレクトリ管理: 初回起動時にホームディレクトリに専用フォルダを自動作成します。🚀 はじめかた1. 動作環境Python 3.8 以上がインストールされていること。2. ライブラリのインストール以下のコマンドを実行して、必要なパッケージをインストールしてください。pip install customtkinter
3. アプリの起動メインスクリプトを実行するとアプリが立ち上がります。python my_notepad_app.py
⌨️ ショートカットキーキー機能Ctrl + N新規タブを作成Ctrl + Oファイルを開くCtrl + Sファイルを保存Ctrl + F検索パネルの表示/非表示Ctrl + ,設定画面の表示/非表示Ctrl + L行番号の表示/非表示を切り替えCtrl + G罫線（グリッド）の表示/非表示を切り替えCtrl + H現在行ハイライトの有効/無効を切り替え⚙️ 設定画面ツールバーの「設定（Settings）」ボタン、または Ctrl + , から以下のカスタマイズが可能です。外観: ダークモード/ライトモードの切り替え。エディタ: フォントサイズの変更、各種視覚ガイド（行番号・罫線・ハイライト）の ON/OFF。ファイル: デフォルトの保存先ディレクトリの指定。言語: 日本語/英語の切り替え（即時適用）。📁 専用フォルダについてユーザーの利便性を高めるため、初回起動時に以下のディレクトリを自動的に作成し、デフォルトの保存先として設定します。パス: ~/ProMultiTabNotepad (ホームディレクトリ直下)🛠 開発者向け情報このプロジェクトは Mixin パターン を採用しており、機能ごとにクラスを分割して管理しています。TabOperationsMixin: タブのライフサイクル管理。FileOperationsMixin: ファイル I/O 処理。SearchOperationsMixin: 検索ロジックと UI 制御。SettingsOperationsMixin: アプリ設定の保持と画面の制御。📄 ライセンスこのプロジェクトは MIT License のもとで公開されています。Produced with passion for clean coding.
